# Aria Voss - Contemporary Artist Portfolio

A clean, minimal, and modern single-page portfolio website showcasing the art, history, and contact information for Berlin-based contemporary mixed-media artist **Aria Voss**.

## Project Overview

This website serves as a digital gallery and profile for Aria Voss. It features a curated selection of oil, acrylic, watercolor, and mixed-media artworks, a biography of the artist, a list of past and upcoming exhibitions, and a direct contact portal.


## File Structure

The project has a simple, static file structure:

 **project.html: The main structure and content of the website. It is semantically organized using HTML5 elements (`<header>`, `<main>`, `<section>`, `<article>`, and `<footer>`) and incorporates modern typography from Google Fonts.

**project.css: The custom styling for the page, containing layouts (using CSS Grid and Flexbox), responsive design queries, colors, and typography styles.

**`bg.png`**: A local background asset used to enhance the visual design of the site.


## Key Features

1.  **Navigation Bar**: A sticky header containing navigation links that smooth-scroll to corresponding sections.
2.  **Hero Section**: An engaging entry section with a bold artist statement: *"Where silence becomes form."*
3.  **Selected Works Gallery**: An asymmetrical CSS Grid layout showing high-quality artwork previews with titles, mediums, sizes, and                        creation years.
4.  **About Section**: A brief bio section featuring studio photography and statistics (years of practice, exhibitions, works created).
5.  **Exhibitions Timeline**: A categorized list highlighting upcoming and past exhibitions in Berlin, London, Hamburg, and Milan.
6.  **Contact Portal**: Simple contact details for inquiries, studio visits, and commissions.
